# Simple FunctionGraph example in C#

This folder contains a simple FunctionGraph function written in C#.

To build execute:

```
dotnet build
```

This will generate a **sample_\<TARGETFRAMEWORK\>.zip** file which can be
uploaded to FunctionGraph as **ZIP Code**.

The handler name of this function is: **simple::src.Program::Handler**

For details see: [Developer Guide for C#](https://opentelekomcloud-community.github.io/otc-functiongraph-csharp-runtime/devguide/event_function/_index.html)
