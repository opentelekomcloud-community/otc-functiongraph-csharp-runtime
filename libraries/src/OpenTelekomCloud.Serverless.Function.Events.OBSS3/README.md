# OpenTelekomCloud.Serverless.Function.Events.OBSS3

This package contains classes that can be used as input types for FunctionGraph that process Object Storage Service (S3) events.

For details, see [OBSS3 Event](https://opentelekomcloud-community.github.io/otc-functiongraph-csharp-runtime/devguide/event_function/trigger_events/trigger_obs_event.html)
