# v2.0.0

## What's changed

### Changes

- [Issue #1](https://github.com/opentelekomcloud-community/otc-functiongraph-csharp-runtime/issues/1): `APIGResponse` added to `APIGEvent`.

- added: `OpenTelekomCloud.Serverless.Function.Common.TestUtils`  
  Provides utility classes for XUnit tests, E.g. TestContextSimple.

- added: `APIGEvent.getBodyAsString()`  
  returns body as string (decodes if necessary)

- Refactorings: Moved libraries sources to src `sub` folder.

### BREAKING CHANGES:

- [Issue #2](https://github.com/opentelekomcloud-community/otc-functiongraph-csharp-runtime/issues/2) : `OpenTelekomCloud.Serverless.Function.Events.DDS` removed as DDS is deprecated.

- [Issue #3](https://github.com/opentelekomcloud-community/otc-functiongraph-csharp-runtime/issues/3) : `APIGEvent`, support added for not predefined properties in `APIGRequestHeaders`, `APIGResponseHeaders`, `QueryStringParameters`, `PathParameters`

- [Issue #4](https://github.com/opentelekomcloud-community/otc-functiongraph-csharp-runtime/issues/4): library `OpenTelekomCloud.Serverless.Function.Common.legacy` removed, now part of `OpenTelekomCloud.Serverless.Function.Common`
